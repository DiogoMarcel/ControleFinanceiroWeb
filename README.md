# Controle Financeiro Web

Sistema de controle financeiro familiar — versão web.

Migração do sistema desktop (Delphi) para aplicação web moderna, responsiva e acessível de qualquer dispositivo.

## Stack

- **Frontend**: React + TypeScript + Tailwind CSS + Recharts
- **Backend**: Node.js + Fastify + Prisma ORM
- **Banco de dados**: PostgreSQL
- **Autenticação**: Firebase Authentication
- **Deploy**: Vercel (frontend) + Railway (backend)

## Documentação

A especificação completa do projeto está em:
`D:\Projetos\IA - Delphi\controleFinanceiro\spec\`

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```
